# REDs图像训练数据集

## 数据集介绍

本资源文件提供了一个用于训练去模糊算法的数据集，名为REDs图像训练数据集。该数据集包含清晰和模糊图片各24000张，涵盖了240个不同的场景。每对图片由一张模糊图片和一张对应的清晰图片组成，总共包含24000对图片。

## 数据集用途

该数据集主要用于训练和评估图像去模糊算法。通过使用这些清晰和模糊的图片对，研究人员和开发者可以开发和优化各种去模糊算法，以提高图像处理的效果。

## 数据集内容

- **清晰图片**：24000张
- **模糊图片**：24000张
- **场景数量**：240个
- **图片对数量**：24000对
